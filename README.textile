h1. Ruxtape: A Ruby MixTape Implementation

Ruxtape is a clone of "opentape":http://opentape.fm, written with Camping and Jquery.  It currently only authenticates using OpenID.

h2. Installing

* @sudo gem install camping@
* @git clone git@github.com:ch0wda/ruxtape.git@
